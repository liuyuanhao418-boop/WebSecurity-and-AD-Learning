# RedTeam Learning Notes

This repository records my learning process in Web Security and Internal Penetration Testing.

The goal of this project is to systematically organize security knowledge and simulate real attack paths used in penetration testing and red team operations.

## Web Security

Research and practice of common web vulnerabilities:

- SQL Injection
- Cross-Site Scripting (XSS)
- File Upload Vulnerability
- CSRF
- Burp Suite testing workflow

## Active Directory Security

Study of common attacks in Windows domain environments:

- NTLM authentication
- Kerberos authentication
- Pass-the-Hash
- NTLM Relay
- ACL abuse
- BloodHound attack path analysis

## Internal Penetration Testing

Simulation of enterprise internal network attack chains:

- Information gathering
- Credential attacks
- Privilege escalation
- Lateral movement

## Tools

Common tools used during learning:

- Burp Suite
- BloodHound
- Mimikatz
- Impacket
- Nmap
- Wireshark

## Learning Method

1. Study security principles
2. Organize knowledge using XMind
3. Practice in security labs (TryHackMe)
4. Reproduce attack chains in a simulated environment
