# 红队学习笔记（RedTeam Learning Notes）

本仓库用于记录我在学习 **Web安全与内网渗透测试** 过程中的知识整理与实践笔记。

项目目标是系统化整理安全知识，并模拟渗透测试与红队攻击中的常见攻击路径。

---

## Web安全

研究与实践常见 Web 漏洞：

- SQL 注入（SQL Injection）
- 跨站脚本攻击（XSS）
- 文件上传漏洞
- CSRF
- Burp Suite 漏洞测试流程

---

## Active Directory 域安全

研究 Windows 域环境中的常见攻击技术：

- NTLM 认证机制
- Kerberos 认证机制
- Pass-the-Hash 攻击
- NTLM Relay 攻击
- ACL 权限滥用
- BloodHound 域关系分析

---

## 内网渗透

模拟企业内网攻击流程：

- 信息收集
- 凭证获取
- 权限提升
- 横向移动
